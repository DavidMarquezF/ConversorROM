==================
ROM to VHDL
==================

Mòdul encarregat de passar de codi ROM a VHDL. S'utiltiza en el mòdul VHDLdePartitura però també es pot crear un fitxer
amb ROM i t'ho passarà a VHDL si s'executa directament aquest mòdul.

Per customitzar/ampliar les instruccions que pot gestionar visiteu l'apartat de Customització.
